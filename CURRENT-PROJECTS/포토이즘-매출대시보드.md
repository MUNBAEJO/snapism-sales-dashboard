# 포토이즘 매출 대시보드 자동화

> 포토이즘(Photoism) 글로벌 어드민 CMS 자동 수집 + Streamlit 실시간 대시보드

---

## 🎯 목적

포토이즘 글로벌 CMS에서 30개국 일별 결제 데이터를 자동 크롤링하여 누적 저장하고,
국가별·브랜드별·타이틀(IP)별 매출을 한 화면에서 모니터링한다.

---

## ✅ 효과

- 매일 09:00 자동 수집 → 수동 CSV 다운로드 불필요
- 30개국 다통화 KRW 환산 실시간 확인
- 국가별 타이틀(IP) TOP 10 피벗 테이블 자동 생성
- 국가별 쿠폰·서비스코인 포함 매출액 자동 산출

### 🔍 필터링·비교 방법

**CMS 매출 데이터**
- `취소 여부 = False` → 취소 건 전체 제외
- `최종 결제 금액 >= 0` → 음수 건 제외
- **다통화 KRW 환산**: `KRW환산 = 최종결제금액 × 환율` (SMBS 매매기준율 기준)

**국가별 매출액 계산 공식**
```
매출액 = 최종결제금액(KRW환산)
       + 쿠폰할인금액(KRW환산)   ← 쿠폰 포함 국가: la, gb, de, th, lv, mx
       + 서비스코인(KRW환산)     ← 서비스코인 포함 국가: cl, la, pe, gb, de, lv, mx
```

### 📊 대시보드 구성

| 섹션 | 내용 |
|---|---|
| KPI 카드 | 오늘/어제/이번달/조회기간 매출 (KRW) |
| 일별 매출 추이 | 바 차트 + 7일 이동평균 라인 |
| 국가별 매출 비중 | 도넛 파이 차트 |
| 국가별 매출 TOP 10 | 가로 바 차트 |
| 매장별 TOP 10 | 가로 바 차트 |
| 타이틀(IP) TOP 10 | 가로 바 차트 + 전체 펼치기 |
| **IP 상세 분석** | 프레임 필터 선택 시 — 단일: 일별 추이/국가 분포; **복수: IP별 기여도 비교 + 합산 추이 + 국가 분포** |
| 국가별 IP TOP 10 | 피벗 테이블 (국가×순위) |
| 시간대별 분포 | 0~23시 바 차트 |
| **세부 판매 항목 검색** | 프레임 이름/테마(구좌)/타이틀/브랜드/중분류/소분류 분류별 집계 + 검색어 필터 + TOP 20 바차트 + CSV (DuckDB on-demand) |
| 원본 데이터 | 필터된 raw 데이터 + CSV 다운로드 |

**사이드바 필터**: 날짜 범위 / 국가 / 매장 / 카테고리(아티스트/캐릭터) / 상품 카테고리 / **프레임 (IP) 멀티셀렉트**

### ✅ 검증 가능 예시

**① CMS 건수 검증**
```
포토이즘 어드민 CMS → 날짜 필터 동일 기간 → 건수 확인
= data/master_photoism.csv 해당 날짜 행수 (취소 포함)
```

**② 매출액 검증 (쿠폰·서비스코인 포함 국가)**
```
예) 영국(gb): 최종결제금액(GBP) + 쿠폰할인금액(GBP) + 서비스코인
    → 대시보드 국가별 TOP 10 영국 매출액과 일치해야 함
```

**③ 환율 검증**
```
SMBS (http://www.smbs.biz/ExRate/TodayExRate.jsp) → 오늘 환율
= 대시보드 상단 "💱 환율 기준" 캡션값
```

### ⚠️ 알려진 제외 케이스

| 케이스 | 처리 방식 |
|---|---|
| **취소 건** | `취소 여부 = True` 전체 제외 |
| **쿠폰·서비스코인 비포함 국가** | 최종결제금액만 집계 (국가 규칙 상이) |
| **CNH (역외 위안화)** | CNY와 동일 환율 적용 (SMBS는 CNH만 제공) |
| **PEN, LAK** | SMBS 미지원 → fawazahmed0 기본값 사용 |
| **SMBS 접속 실패 시** | fawazahmed0 API로 자동 fallback |
| **환율 캐시 (과거 날짜)** | 한 번 조회 후 영구 캐시. 수정 필요 시 `data/rates_date_cache.json` 해당 날짜 삭제 |

## ⚙️ 동작 방식

1. Windows 작업 스케줄러(`snapism_daily`)가 매일 09:00 `photoism_crawler.py` 자동 실행
2. Playwright가 30개국 CMS에 순차 로그인 → 전날 날짜로 필터 → CSV 자동 다운로드
3. `ingest.py`가 raw CSV를 `data/master_photoism.csv`에 누적 합산 (날짜+주문번호 기준 중복 제거)
4. SMBS 환율 자동 갱신 → KRW 환산 반영
5. Streamlit 대시보드(포트 8503) `1_📸_포토이즘` 페이지에서 실시간 조회
